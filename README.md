# AppTemplate

Base on @angular/cli v6.0.7

# App includes:

- Angular v6
- Material Design
- Bootstrap 4 
- Translate Module 
- Translate Component,Service and Pipe
- Router Module
- Log Service
- Auth Interceptor

# Installation

1. Clone this rep.
2. npm install

# File structure:

|- src/
    |- assets
    |- components
        |- language(Component)
    |- directives
    |- environments
    |- interceptors
        |- auth-interceptor
    |- lang
        |- lang-en.ts
        |- lang-he.ts
    |- modeles
        |- log.model.ts
    |- modules
        |- material(Module)
        |- router(Module)
        |- translate(Module)
    |- pages
        |- app(Component)
    |- pipes
    |- services
        |- log(service)
        |- storage(service)
